🚀 NewAPI 配置生成器

轻松生成多账号签到配置，同站点多账号只需填写一次 URL

☁️ 云端同步

将配置保存到你的云盘/NAS，GitHub Actions 签到脚本可直接从云端拉取配置，改配置无需推代码

WebDAV 服务地址，末尾需包含文件路径
坚果云请在账户设置中创建「应用密码」
云盘上的配置文件名（JSON 格式）

💡 云端同步说明：

  • 坚果云：使用 WebDAV 协议，需在账户设置中创建「应用专用密码」（不是登录密码）
  • 群晖 NAS：需启用 WebDAV 服务（端口 5005/5006），确保 NAS 可从外网访问
  • NextCloud：地址格式为 https://域名/remote.php/dav/files/用户名/
  • GitHub Actions：保存后，将生成的 CONFIG_URL 和 CONFIG_AUTH 添加到仓库 Secrets
  • CORS 限制：浏览器直接访问可能受 CORS 限制，如遇跨域错误请参考下方解决方案

📋 导入 JSON 配置

粘贴已有的 JSON 配置，一键导入并按站点自动分组填充表单

💡 使用提示：

  • 站点分组：同一站点的 URL 和 CF Clearance 只需填写一次，站点下可添加多个账号
  • 账号信息：每个账号只需填写 Session、用户ID 和备注名称
  • 保存到本地：将配置保存到浏览器本地存储，下次访问自动加载
  • 明文存储：数据以明文形式存储在浏览器中，请注意安全
